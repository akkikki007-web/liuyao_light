# 六爻装卦ツール

六数字または卦名指定から六爻の装卦表を作る静的 Web アプリです。

`index.html` をブラウザで開くか、このフォルダをローカルサーバーで配信して使います。

## 現在の仕様

- 六数字入力に対応
  - 6つの数字を下から順に、初爻、二爻、三爻、四爻、五爻、上爻として読む
  - 使う数字は1〜8
- 卦名指定に対応
  - 本卦と之卦を選択できる
  - 之卦は「不動」も選べる
- 数字と陰陽
  - 1: 老陽
  - 2: 少陰
  - 3: 少陰
  - 4: 少陽
  - 5: 少陰
  - 6: 少陽
  - 7: 少陽
  - 8: 老陰
  - 老陽と老陰を動爻として之卦に反映
- 6つの数字が揃うまで
  - 入力済みの爻だけを表示
  - 卦名、六親などの装卦表、爻同士の関係は非表示
- 元アプリのCanvas帳票を維持
  - 卦の記号表示
  - 本卦、之卦の表示
  - 装卦表の形
  - 伏爻、本爻、変爻の列
- 八宮、世応、納甲、六親を表示
- 日干から起こした六神を表示
- 月建・日辰による生、剋、合、冲、空亡を表示
- 爻同士の生、剋、合、冲を表示

## 削除した機能

- 占例データベース
- 占例の保存、読み込み、削除、検索
- 占例JSONの書き出し、読み込み
- 保存フォルダ指定
- `liuyao-cases.json` との同期保存
- 占的欄とメモ欄
- PNG、Word、PDF保存ボタン
- テキスト出力とコピー
- 印刷

## 注意

六爻は流派差が出やすいので、数字列から爻への読み替え、世応、納甲、六親の対応は `app.js` のテーブルで明示しています。使っている流派の表がある場合は、該当テーブルを差し替えると合わせられます。
